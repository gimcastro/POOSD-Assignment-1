#Colors LAMP Lab Web Application 

#Description 
Colors is a web application built with LAMP stack. The application was built to allow users to log in and manage their colors added to the database. 

#Technologies Used
Linux
MySQL
PHP
HTML
CSS
JavaScript

#Setup Instructions
Install Apache, MySQL and PHP
Place files inside "/var/www/html"
Create MySQL database named "DB_NAME" 
Access application via browser using server IP

#How to run
Navigate: 
http://YOUR_SERVER_IP/index.html
